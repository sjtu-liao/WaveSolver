# WaveSolver

The source code, training scripts, and model configurations will be released in this repository after publication. 

At this stage, we provide the public dataset sources used in the experiments to support reproducibility.

## Public Datasets

| Dataset       | Data source                                                  |
| ------------- | ------------------------------------------------------------ |
| Airfoil       | [Geo-FNO standard benchmark data](https://drive.google.com/drive/folders/1YBuaoTdOSr_qzaow-G-iwvbUI7fiUzu8) |
| Pipe          | [Geo-FNO standard benchmark data](https://drive.google.com/drive/folders/1YBuaoTdOSr_qzaow-G-iwvbUI7fiUzu8) |
| Darcy         | [FNO standard benchmark data](https://drive.google.com/drive/folders/1UnbQh2WWc6knEHbLn-ZaXrKUZhp7pjt-) |
| Navier-Stokes | [FNO standard benchmark data](https://drive.google.com/drive/folders/1UnbQh2WWc6knEHbLn-ZaXrKUZhp7pjt-) |
| ShapeNet-Car  | [ShapeNet-Car CFD data](http://www.nobuyuki-umetani.com/publication/mlcfd_data.zip) |
| AirfRANS      | [AirfRANS dataset](https://data.isir.upmc.fr/extrality/NeurIPS_2022/Dataset.zip) |

## Acknowledgements

We gratefully acknowledge the authors and maintainers of the public benchmark datasets and open-source resources used in this work, including FNO, Geo-FNO, ShapeNet-Car CFD data, and AirfRANS. These public datasets and reference implementations provide an important basis for reproducible research and fair comparison in neural PDE surrogate modeling.

Users of this repository should also cite the original papers and dataset sources when using these datasets.
